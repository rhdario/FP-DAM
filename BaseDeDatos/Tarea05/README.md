# Tarea 05: Consultas y Scripting SQL

Este directorio contiene la solución a la Tarea 5 de la asignatura de **Base de Datos**.

## 📁 Archivos incluidos

* **`classicmodels.sql`**: Script original con la estructura y datos iniciales de la base de datos.
* **`solucionclassicmodels.sql`**: Script unificado que contiene todas las sentencias y comandos SQL requeridos en la solución de la tarea.

## 🛠️ Instrucciones
Para ejecutar la solución, se recomienda cargar primero el archivo original y posteriormente ejecutar el script de soluciones.
